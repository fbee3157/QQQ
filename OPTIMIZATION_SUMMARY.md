## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置密钥
```bash
cp .env.example .env
# 编辑 .env，填入真实 API 密钥
```

### 3. 启动系统
```bash
# 方式1：分离启动
python live_trader.py
python trader_web.py

# 方式2：守护模式
python watchdog.py
```

### 4. 访问仪表盘
打开浏览器访问 `http://127.0.0.1:8080`

## 🎓 调优建议

### 信号灵敏度
```python
# 交易机会少 → 提高灵敏度
"vol_mult": 0.6  # 降低量能要求
"min_body": 0.0001  # 降低 K 线实体要求

# 假信号多 → 降低灵敏度
"vol_mult": 1.2  # 提高量能要求
"min_body": 0.0005  # 提高 K 线实体要求
```

### 止盈风格
```python
# 激进风格（高频小利）
"tp_partial_pct": 0.30  # 30% 利润平一半
"tp_trail_drop": 0.15  # 15% 回撤平仓

# 保守风格（低频大利）
"tp_partial_pct": 1.50  # 150% 利润平一半
"tp_trail_drop": 0.40  # 40% 回撤平仓
```

## ⚠️ 注意事项

1. **密钥安全**
   - `.env` 绝不能提交到 Git
   - `.gitignore` 已配置自动忽略
   - 定期轮换 API 密钥

2. **交易窗口**
   - 美东 09:35-15:50（避免开盘/收盘噪音）
   - 不支持盘前/盘后交易
   - 美股期权 0DTE 专用

3. **资金管理**
   - 每笔 10 张期权起
   - 日最多 8 笔交易
   - 日亏损 5% 自动停止

4. **系统要求**
   - Python 3.10+
   - Linux/WSL 推荐（Windows 原生不稳定）
   - 稳定网络连接
