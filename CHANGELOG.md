## 升级指南

### 从 v6.0 升级到 v6.1

1. **备份现有数据**
   ```bash
   cp -r records/ records_backup/
   cp state.json state_backup.json
   ```

2. **更新代码**
   ```bash
   git pull origin main
   # 或直接复制新文件覆盖
   ```

3. **更新依赖**
   ```bash
   pip install -r requirements.txt --upgrade
   ```

4. **检查 .env 文件**
   - 现有 `.env` 完全兼容，无需修改
   - 可选新增：`TRADE_SYMBOL`, `GITHUB_TOKEN`, `GIST_ID`

5. **启动测试**
   ```bash
   python live_trader.py  # 测试信号生成
   python trader_web.py   # 测试 Web 页面
   ```
